## Flags

### `lea_flt_size`

Integer. Set the filter buffer size on LEA's SRAM

### `lea_src_size`

Integer. Set the source buffer size on LEA's SRAM

### `lea_dst_size`

Integer. Set the destination buffer size on LEA's SRAM

### `dma_opt`

Boolean. Enable DMA optimization

### `lea_opt`

Boolean. Enable LEA optimization

### `adaptive_gen_lea`

Boolean. Enable Adaptive generation of LEA accelerations
